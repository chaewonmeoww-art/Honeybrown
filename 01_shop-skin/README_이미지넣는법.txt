MAISON 쇼핑몰 스킨 — 이미지 적용 안내
=======================================

[중요] 이 폴더의 HTML/CSS/JS는 모두 완성되어 있습니다.
다만 이 작업 환경의 제약으로 "이미지 파일(.png)"만 디스크에 직접 저장하지 못했습니다.
아래 방법으로 이미지만 넣으면 바로 완성된 화면이 보입니다.

■ 넣는 방법
1) 함께 전달한 "maison-images.zip" 파일의 압축을 풉니다.
2) 안에 들어있는 "images" 폴더를 이 폴더(01_shop-skin) 안에 그대로 복사합니다.
   → 최종 구조:  01_shop-skin / images / hero1.png, item01.png ...
3) index.html 을 브라우저로 열면 끝.

■ 들어있는 이미지 목록 (총 26장)
- hero1.png ~ hero3.png      : 메인 히어로 슬라이드   (2000 x 900)
- banner1.png, banner2.png   : 띠배너 2개            (2000 x 500)
- cat_outer / cat_knit / cat_dress / cat_bag .png : 카테고리 썸네일 (800 x 800)
- item01.png ~ item16.png    : 상품 썸네일 16장       (600 x 600)
- detail_long.png            : 상품상세 설명 이미지   (800 x 3000)

■ 실사 사진으로 교체하기
지금 들어있는 이미지는 디자인 확인용으로 그린 "편집형 일러스트 시안"입니다.
(이 환경에서는 실제 사진 촬영/생성이 불가능합니다.)
같은 파일명 / 같은 크기로 실제 촬영 사진을 덮어쓰면 디자인이 깨지지 않습니다.
  · 상품 썸네일은 정사각 600x600 권장 (목록에서는 축소 표시됨)
  · 상세 설명 이미지는 폭 800, 높이 자유 (예: 800x3000)
  · 히어로는 2000x900, 띠배너는 가로로 긴 비율

■ 색/폰트/간격 등 손쉽게 수정
각 HTML 파일 <head> 안의  :root { ... } 변수만 바꾸면 전체에 반영됩니다.
  --point  : 포인트 색 (버튼/가격/배지/링크호버) — 한 곳만 바꾸면 전부 적용
  --main / --sub / --bg / --bg-soft / --line : 메인·보조·배경·라인 색
  --radius : 모서리 둥글기 (현재 8px)
  --gap-section / --pad-x / --maxw : 섹션 간격 / 좌우 여백 / 최대 폭(1550)
  --fs-* / --fw-* : 계층별 글자 크기 / 두께

■ 페이지 구성
  index.html          메인 (히어로/카테고리/추천/신상품/베스트/띠배너2/푸터, 각 <section>)
  products.html       상품목록 (정렬바 + 카드 16개 + 페이지네이션)
  product-detail.html 상품상세 (이미지 + 옵션/수량/구매 + 상세이미지 + 상세/리뷰/문의 탭)
  cart.html           장바구니
  login.html / signup.html  로그인 / 회원가입
  board.html / board-view.html  게시판 목록 / 보기

※ 구매·결제·로그인 "기능"은 넣지 않았습니다(카페24가 처리). 화면(디자인)만 구현했습니다.
